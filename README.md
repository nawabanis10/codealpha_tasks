# codealpha_tasks

🤖 Python Chatbot 
📌 Description

This is a simple rule-based chatbot built using Python.
The chatbot can interact with users, respond to basic queries, tell jokes, and provide current time and date.

It uses predefined responses and basic keyword matching to simulate a conversation.

🚀 Features
👋 Greeting responses (hi, hello, hey)
😊 Basic conversation (How are you)
😂 Joke generator
⏰ Current time display
📅 Current date display
🙏 Thanks response
📛 Bot name response
❓ Help command
🧠 Handles unknown inputs
📝 Conversation history stored in a file


🛠️ Technologies Used
Python 🐍
random module (for random responses)
datetime module (for time & date)

│── chatbot.py
│── history.txt
│── README.md

2.**🎮 Hangman Game in Python**
📌 Description

This is a simple Hangman game built using Python.
The player has to guess a hidden word by entering one letter at a time.

If the guessed letter is correct, it is revealed in the word.
If the guess is wrong, the player loses a chance.

The player wins if they guess the word before running out of attempts.

🚀 Features

🎯 Random word selection
🔤 Letter-by-letter guessing
❌ Wrong guess tracking
🔢 Limited attempts (6 chances)
🏆 Win/Lose system
📺 Real-time word display

🛠️ Technologies Used
Python 🐍
random module

hangman/
│── hangman.py
│── README.md
